Keystatic is a content management tool built for developers who want a Git-based workflow without
giving up an editing UI.

### How does it work? eine kleine Änderung

Keystatic reads and writes content as files — Markdown, MDX, JSON, or YAML — directly in your
repository. Editors get a polished admin interface at `/keystatic`, while developers keep full
control over schemas, rendering, and deployment.

### Do I need a database?

No. All content lives in your repo. In local mode, changes write straight to disk. In GitHub mode,
every edit creates a pull request so content goes through the same review process as code.

### What frameworks does it support?

Keystatic works with **Astro**, **Next.js**, and **Remix** out of the box. Any framework that can
read local files can integrate with Keystatic's reader API.
